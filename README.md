# PKU-OOP-World-Of-Warcraft-Simulation
这是我在北京大学大一《程序设计实习》课程中完成的魔兽世界大作业
魔兽世界1（WOW_1)实现了基本的司令部生产武士功能
  在魔兽1中我保留了两版
  第一版我创建了一个Warriors类试图通过它来操控所有武士和司令部的行为，但结构过于混乱
  第二版我创建了Headquarter类来操控司令部生产武士的所有逻辑，较上一版结构清晰、可读性强
魔兽世界2（WOW_2)实现了司令部生产武士并配备装备和一些特性输出功能
  魔兽2中我依然保留了两版
  第一版我使用了一个极为庞大的switch语句来实现生产不同武士的功能，个人认为有些太臃肿了
  第二版是在我学习多态后修改的，使用多态相关的知识虽然我依旧用switch语句来控制核心功能，但代码清爽了很多
魔兽世界3(WOW_3)实现了更为复杂的生产武士并配备装备，武士前进、战斗、缴获武器、死亡等一系列功能
  魔兽3是我目前耗时最久的一份作业，尤其是在防治内存溢出和控制指针的时候耗费了我大量心神
  我创建了World类来统筹一局游戏的进行，个人认为是一个非常不错的思路，能使程序逻辑非常清晰
